# 实现网络聊天室

## 一、概述

### **功能分析**

1. 上下线
2. 聊天，其他人和自己都可以看到聊天消息
3. 查询当前聊天室用户名
4. 用户可以修改自己的名称
5. 挂机超时踢出机制

### **技术分析**

1. socket TCP 编程
2. map 结构
   1. 存储所有用户
   2. map 遍历
   3. map 删除
3. goroutine、channel
4. select 实现超时退出、主动退出
5. timer 定时器

## 二、实现基础

### 第一阶段

1. 思路分析
   - TCPsocket 建立多个连接
2. 定义 User 结构
3. 定义 map 结构
4. 定义 message 通道
5. user 监听通道 goroutine
